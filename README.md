# JSTechnoHub - Backend API

Professional Spring Boot API powering the JSTechnoHub Coaching Institute platform.

## 🚀 Tech Stack

- **Framework:** Spring Boot 3.5.4
- **Language:** Java 17
- **Database:** MySQL
- **Build Tool:** Maven
- **Payments:** Razorpay Java SDK
- **Utilities:** Lombok, Spring Data JPA, Java Mail Sender

## 🛠️ Features

- Course Management API
- Razorpay Payment Gateway Integration
- Automated Email Notifications (Inquiry & Enrollment)
- Data Validation and Global Error Handling

## 🔧 Local Setup

1. Clone the repository.
2. Update `src/main/resources/application.properties` with your MySQL credentials.
3. Run `./mvnw spring-boot:run`.
